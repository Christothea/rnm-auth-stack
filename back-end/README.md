# Registration and Authentication backend

## Environment
- NodeJs
- Typescript
- NestJs
- Swagger
- MySql

## Setup Instructions
1. Start MySql server
2. Install Node
3. npm install
4. Create .env file and copy the contents of the .env.example, fill the values accordingly
5. npm start
