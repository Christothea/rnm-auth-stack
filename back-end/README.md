# # Registration and Authentication backend

## Environment
- NodeJs
- Typescript
- NestJs
- Swagger
- MySql

## Setup Instructions
1. Start MySql server
2. npm install
3. Create .env file and copy the contents of the .env.example, fill the values accordingly
4. npm start
